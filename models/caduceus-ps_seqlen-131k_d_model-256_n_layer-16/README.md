# Local checkpoint placeholder

Put the pretrained weights for `caduceus-ps_seqlen-131k_d_model-256_n_layer-16` in this directory before training.
The original project used local HuggingFace-style folders. Large weight files such as
`model.safetensors`, `pytorch_model.bin`, or equivalent checkpoint shards are intentionally
not included in this GitHub-ready release.
